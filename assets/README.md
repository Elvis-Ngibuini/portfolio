# Assets Directory

This directory contains all the static assets for the portfolio website.

## Image Requirements

### Profile Picture
- **File**: `profile.jpg` (or `profile.png`)
- **Dimensions**: At least 500x500 pixels (1:1 aspect ratio)
- **Style**: Professional headshot with a clean background
- **Naming**: Save as `profile.jpg` (or `profile.png`)

### Project Thumbnails
- **File Naming**: Use descriptive names like `huduma-dashboard.jpg`, `chatbot.jpg`, etc.
- **Dimensions**: 800x450 pixels (16:9 aspect ratio)
- **Style**: Clean screenshots or mockups of your projects

### Icons
- All icons are loaded from Font Awesome CDN
- No need to add any icon files manually

### How to Add Images
1. Place your profile picture in this directory as `profile.jpg` (or `profile.png`)
2. Add project thumbnails with descriptive names
3. Update the image references in the HTML file if you use different filenames

## Recommended Tools for Images
- [TinyPNG](https://tinypng.com/) - For image compression
- [Canva](https://www.canva.com/) - For creating project mockups
- [Remove.bg](https://www.remove.bg/) - For removing image backgrounds
