Manual Backup Instructions

For a complete backup, follow these steps:

  1. Copy Portfolio Folder: Copy your entire portfolio folder to a safe location (external drive, cloud storage)
  2. Export Contact Data: Use the admin panel to export contact submissions as CSV
  3. Save Configuration: Keep a copy of your portfolio-config.js and server/.env files
  4. Document Changes: Keep notes of any customizations you've made
  5. Test Backup: Periodically test that your backup works by setting it up in a new location

Recommended: Set up automatic backups using Git or cloud storage sync.
