How to Start Your Servers

  1. Open Command Prompt/Terminal in your portfolio folder
  2. Start Frontend Server:
    python -m http.server 8000
  3. Open Another Terminal and navigate to the server folder
  4. Start Backend Server:
    cd server
    npm run dev
  5. Wait for both servers to show as online above
  6. Click the launch buttons to access your portfolio

Alternative: Run start-dev.bat to start both servers automatically.
