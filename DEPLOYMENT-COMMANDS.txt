2. Enable 2-Step Verification first
3. Create app password for "Mail"
4. Copy 16-character password
5. Use in EMAIL_PASS environment variable

üÜò TROUBLESHOOTING
================================================================================

Contact form not working:
   - Check browser console (F12)
   - Verify backend URL in js/api.js
   - Wait 30 seconds (backend may be sleeping)

Admin panel authentication failed:
   - Verify API key matches in admin.html and backend
   - Check browser console for 401 errors

Emails not received:
   - Check spam folder
   - Verify Gmail App Password (16 characters)
   - Check Render logs for errors
