Content Updates

Update Profile Image

Change your main profile photo that appears in the hero section

Current: images/eluis.jpg
  1. Add your new image to the images/ folder
  2. Name it something descriptive (e.g., profile-2025.jpg)
  3. Update the image path in index.html
  4. Test the changes

Update Personal Information

Change your name, title, description, and contact details

File: index.html (lines 50-80)
  1. Open index.html in a text editor
  2. Find the hero section
  3. Update name, title, and description
  4. Update contact information in the contact section

Add New Project

Add a new project to your portfolio with images and descriptions

File: index.html (Projects Section)
  1. Add project image to images/ folder
  2. Copy an existing project card in index.html
  3. Update the project details
  4. Add appropriate tech stack tags

Update Skills & Technologies

Add new skills, update proficiency levels, or reorganize tech stack

File: index.html (Skills & Tech Stack Sections)
  1. Find the skills or tech-stack section
  2. Add new skill items or update existing ones
  3. Update proficiency percentages
  4. Add appropriate icons from Font Awesome

Add Work Experience

Add new job positions, internships, or update existing experience

File: index.html (Experience Section)
  1. Find the timeline section in experience
  2. Copy an existing timeline-item
  3. Update job title, company, dates
  4. Add achievements and tech tags

Add Challenges/Certifications

Add new security challenges, certifications, or learning achievements

File: index.html (Challenges Section)
  1. Add certificate image to images/ folder
  2. Copy an existing challenge-card
  3. Update challenge details and learnings
  4. Add appropriate tools and categories
