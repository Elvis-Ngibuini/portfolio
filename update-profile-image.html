Update Profile Image

Easy guide to change your portfolio profile photo

Current Profile Image

Location: images/eluis.jpg
Recommended Size: 400x400px
Format: JPG, PNG, WebP
Max File Size: 200KB

Quick Update Steps

  1. Optimize your image
    Use the Image Optimizer tool
  2. Save as eluis.jpg
    Or use a different name
  3. Copy to images folder
    Replace the existing file
  4. Refresh browser
    Press Ctrl+F5 to see changes
Open Image Optimizer

Method 1: Replace Existing Image (Easiest)

Recommended Method
This method doesn't require editing any code!
  1. Prepare your new image
    • Make it square (400x400px recommended)
    • Use JPG format for photos
    • Keep file size under 200KB
  2. Optimize the image
    Optimize Image
  3. Rename to eluis.jpg
    This way you don't need to edit any code
  4. Navigate to images folder
    C:\Users\USER\OneDrive\Desktop\PROJECTS\portfolio\images\
  5. Replace the file
    Backup the old eluis.jpg first if you want to keep it
  6. Test your changes
    View Portfolio

Method 2: Use Different Filename

Requires Code Editing
Use this method if you want to keep the old image or use a different filename.
  1. Add your new image to images folder
    Example: profile-2026.jpg
  2. Open index.html in code editor
    Find line 80 (approximately)
  3. Find this line:
    <img src="images/eluis.jpg" alt="Eluis Ngibuini Thamaini" class="profile-img">
  4. Change to your new filename:
    <img src="images/profile-2026.jpg" alt="Eluis Ngibuini Thamaini" class="profile-img">
  5. Save and refresh browser
    Press Ctrl+F5 to clear cache

Troubleshooting

Image Not Showing

  • Check file is in images/ folder
  • Verify filename matches exactly
  • Clear browser cache (Ctrl+F5)
  • Check file format (JPG, PNG, WebP)

Image Too Large

  • Use Image Optimizer tool
  • Reduce quality to 80-85%
  • Resize to 400x400px
  • Convert to JPG format

Image Looks Blurry

  • Use higher quality original
  • Increase quality to 90%
  • Ensure original is 400x400px+
  • Don't upscale small images
